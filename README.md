# axes

`axes` is a utility-library for Computer Graphics.

Build System Requirement:

- gcc
- clang: 15.0

# Dependencies

Arch Linux:

```
yay -S libcap 
yay -S libcap gperf libsystemd
yay -S libmount
yay -S liblzma
yay -S clang15
yay -S xz dbus
```
